PROJECT OVERVIEW
Image Recognition System
using Deep Learning

A complete end-to-end deep learning pipeline for detecting and classifying objects from images. The system uses MobileNetV2 pretrained on ImageNet, served via a Flask REST API with a responsive frontend.

Production Ready REST API Deep Learning
OBJECTIVES
1
Object Detection & Classification
Identify objects across 1000+ categories
2
Real-time Inference
Sub-20ms predictions via optimized model
3
REST API Backend
Flask server with CORS & file handling
4
Interactive Dashboard
Live pipeline, charts, confidence bars
HOW TO RUN
# 1. Install dependencies
pip install -r requirements.txt
# 2. (Optional) Train custom model
python train_model.py
# 3. Start backend server
python app.py
# Runs on http://localhost:5000
# 4. Open frontend
Open index.html in your browser
